# рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ

Laravel -рЃЌрЃљрЃю рЃАрЃљрЃЏрЃБрЃерЃљрЃЮрЃЊ рЃЊрЃљрЃњрЃЋрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ: рЃарЃљ рЃЌрЃЦрЃЏрЃљ рЃБрЃюрЃЊрЃљ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃў ­Ъўѓ рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ рЃбрЃћрЃЦрЃАрЃб-рЃћрЃЊрЃўрЃбрЃЮрЃарЃў рЃљрЃю IDE (Integrated Development Environment - рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃў рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃ░рЃДрЃЮрЃцрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃАрЃљрЃА рЃЊрЃљ рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃћрЃарЃЌрЃўрЃљрЃю рЃњрЃарЃљрЃцрЃўрЃЎрЃБрЃџ рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃерЃў (GUI - graphical user interface) рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃАрЃбрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃ«рЃћрЃџрЃАрЃљрЃгрЃДрЃЮрЃћрЃЉрЃАрЃљ рЃЊрЃљ рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА. рЃЏрЃљрЃњ: NetBeans, PhpStorm ...), PHP рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃљрЃбрЃЮрЃарЃў рЃЊрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃАрЃћрЃарЃЋрЃћрЃарЃў.

рЃЊрЃўрЃЊрЃў рЃљрЃџрЃЉрЃљрЃЌрЃЮрЃЉрЃўрЃЌ, Laravel-рЃўрЃА рЃерЃћрЃАрЃгрЃљрЃЋрЃџрЃўрЃАрЃљрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃБрЃЎрЃЋрЃћ рЃћрЃЦрЃюрЃћрЃЉрЃљ рЃерЃћрЃ«рЃћрЃЉрЃљ PHP-рЃА рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃљрЃбрЃЮрЃарЃЌрЃљрЃю рЃЊрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃАрЃћрЃарЃЋрЃћрЃарЃЌрЃљрЃю, рЃћрЃА рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА рЃарЃЮрЃњрЃЮрЃарЃф рЃфрЃљрЃџрЃЎрЃћ рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃћрЃЉрЃў, рЃЏрЃљрЃњ: рЃЋрЃћрЃЉ-рЃАрЃћрЃарЃЋрЃћрЃарЃў Apache, PHP рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃљрЃбрЃЮрЃарЃў, рЃЏрЃЉ рЃАрЃћрЃарЃЋрЃћрЃарЃў MYSQL, рЃљрЃАрЃћрЃЋрЃћ рЃњрЃљрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃБрЃџрЃў рЃюрЃљрЃЎрЃарЃћрЃЉрЃћрЃЉрЃў рЃЏрЃљрЃњ: Denver, OpenServer, XAMPP, WAMP, LAMP рЃЊрЃљ рЃљ.рЃе. рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЏрЃљрЃЌрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃф.

## рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ Composer-рЃўрЃЌ

Composer-рЃў рЃљрЃарЃўрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЏрЃћрЃюрЃћрЃ»рЃћрЃарЃў, рЃарЃЮрЃЏрЃџрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌрЃљрЃф рЃ«рЃЊрЃћрЃЉрЃљ, рЃљрЃЏрЃљ рЃЌрЃБ рЃўрЃЏ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃўрЃА, рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃўрЃА рЃўрЃюрЃбрЃћрЃњрЃарЃўрЃарЃћрЃЉрЃљ PHP рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃљрЃерЃў. Windows рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃБрЃџ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃерЃў Composer-рЃўрЃА рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ рЃАрЃљрЃЎрЃЏрЃљрЃЮрЃЊ рЃЏрЃљрЃарЃбрЃўрЃЋрЃўрЃљ, рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЊрЃЏрЃЮрЃЋрЃгрЃћрЃарЃЮрЃЌ рЃАрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃЮ рЃцрЃљрЃўрЃџрЃў рЃЊрЃљ рЃЊрЃљрЃЋрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃЮрЃЌ (рЃљрЃЋрЃбрЃЮрЃарЃћрЃЉрЃў: Nils Adermann рЃЊрЃљ Jordi Boggiano).
рЃўрЃюрЃбрЃљрЃџрЃљрЃфрЃўрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў рЃЏрЃЮрЃњрЃЋрЃЌрЃ«рЃЮрЃЋрЃА рЃњрЃќрЃљрЃА PHP-рЃЏрЃЊрЃћ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃБрЃарЃћрЃЉрЃЮрЃЊрЃћрЃА рЃљрЃАрЃћ:

![#](https://www.pascallandau.com/img/php7-with-xdebug-2-4-for-phpstorm-on-windows-10/composer/choose-php-version.PNG)

рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљрЃЌрЃљ рЃЋрЃћрЃџрЃўрЃЊрЃљрЃю (cmd) рЃњрЃљрЃЋрЃБрЃерЃЋрЃљрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ:
```
composer
```

рЃЌрЃБ рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЋрЃўрЃ«рЃўрЃџрЃљрЃЋрЃЌ рЃљрЃАрЃћрЃЌ рЃАрЃБрЃарЃљрЃЌрЃА: 

![#](https://res.cloudinary.com/practicaldev/image/fetch/s--Fkol8vJR--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i1.wp.com/fossnaija.com/wp-content/uploads/2020/02/composer-launch-output.png%3Fresize%3D665%252C388%26ssl%3D1)

рЃћрЃАрЃћ рЃўрЃњрЃў рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃгрЃћрЃАрЃарЃўрЃњрЃерЃўрЃљ рЃЊрЃљ рЃЊрЃарЃЮрЃљ рЃњрЃљрЃЋрЃљрЃњрЃарЃФрЃћрЃџрЃЮрЃЌ

## рЃљрЃ«рЃљрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ

рЃљрЃ«рЃљрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ composer-рЃўрЃА рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ `create-project` рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљрЃАрЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ рЃљрЃАрЃћрЃЋрЃћ рЃБрЃюрЃЊрЃљ рЃЏрЃўрЃЋрЃБрЃЌрЃўрЃЌрЃЮрЃЌ рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃў рЃърЃљрЃЎрЃћрЃбрЃўрЃА рЃЊрЃљрЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃљ рЃЊрЃљ рЃўрЃЏ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃЊрЃљрЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃљ рЃАрЃљрЃЊрЃљрЃф рЃњрЃЋрЃАрЃБрЃарЃА, рЃарЃЮрЃЏ рЃерЃћрЃўрЃЦрЃЏрЃюрЃљрЃА рЃљрЃ«рЃљрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃў:

```bash
composer create-project laravel/laravel рЃАрЃљрЃ«рЃћрЃџрЃў
```

рЃЏрЃљрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃарЃљрЃф рЃњрЃљрЃћрЃерЃЋрЃћрЃЉрЃљ рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃўрЃА рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ, Composer-рЃў рЃЏрЃўрЃљрЃЎрЃўрЃЌрЃ«рЃљрЃЋрЃА рЃарЃљрЃдрЃљрЃф рЃгрЃДрЃљрЃарЃЮрЃА ([packagist.org](https://packagist.org)), рЃарЃћрЃАрЃБрЃарЃАрЃА, рЃАрЃљрЃЊрЃљрЃф рЃњрЃљрЃюрЃЌрЃљрЃЋрЃАрЃћрЃЉрЃБрЃџрЃўрЃљ рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃў рЃърЃљрЃЎрЃћрЃбрЃўрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃўрЃАрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃў, рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃў рЃЊрЃљ рЃљ.рЃе рЃЊрЃљ рЃљрЃЏ рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃА рЃњрЃљрЃЊрЃЏрЃЮрЃгрЃћрЃарЃА рЃЕрЃЋрЃћрЃюрЃА рЃџрЃЮрЃЎрЃљрЃџрЃБрЃа рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃерЃў.

рЃџрЃљрЃарЃљрЃЋрЃћрЃџрЃўрЃА рЃцрЃљрЃўрЃџрЃБрЃарЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃЏрЃЮрЃЏрЃЊрЃћрЃЋрЃюрЃЮ рЃюрЃљрЃгрЃўрЃџрЃерЃў рЃњрЃљрЃЋрЃўрЃЋрЃџрЃўрЃЌ рЃљрЃАрЃћ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃБрЃарЃћрЃЉрЃљ:
