# рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ

Laravel -рЃЌрЃљрЃю рЃАрЃљрЃЏрЃБрЃерЃљрЃЮрЃЊ рЃЊрЃљрЃњрЃЋрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ: рЃарЃљ рЃЌрЃЦрЃЏрЃљ рЃБрЃюрЃЊрЃљ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃў ­Ъўѓ рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ рЃбрЃћрЃЦрЃАрЃб-рЃћрЃЊрЃўрЃбрЃЮрЃарЃў рЃљрЃю IDE (Integrated Development Environment - рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃў рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃ░рЃДрЃЮрЃцрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃАрЃљрЃА рЃЊрЃљ рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃћрЃарЃЌрЃўрЃљрЃю рЃњрЃарЃљрЃцрЃўрЃЎрЃБрЃџ рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃерЃў (GUI - graphical user interface) рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃАрЃбрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃ«рЃћрЃџрЃАрЃљрЃгрЃДрЃЮрЃћрЃЉрЃАрЃљ рЃЊрЃљ рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА. рЃЏрЃљрЃњ: NetBeans, PhpStorm ...), PHP рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃљрЃбрЃЮрЃарЃў рЃЊрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃАрЃћрЃарЃЋрЃћрЃарЃў.

рЃЊрЃўрЃЊрЃў рЃљрЃџрЃЉрЃљрЃЌрЃЮрЃЉрЃўрЃЌ, Laravel-рЃўрЃА рЃерЃћрЃАрЃгрЃљрЃЋрЃџрЃўрЃАрЃљрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃБрЃЎрЃЋрЃћ рЃћрЃЦрЃюрЃћрЃЉрЃљ рЃерЃћрЃ«рЃћрЃЉрЃљ PHP-рЃА рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃљрЃбрЃЮрЃарЃЌрЃљрЃю рЃЊрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃАрЃћрЃарЃЋрЃћрЃарЃЌрЃљрЃю, рЃћрЃА рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА рЃарЃЮрЃњрЃЮрЃарЃф рЃфрЃљрЃџрЃЎрЃћ рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃћрЃЉрЃў, рЃЏрЃљрЃњ: рЃЋрЃћрЃЉ-рЃАрЃћрЃарЃЋрЃћрЃарЃў Apache, PHP рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃљрЃбрЃЮрЃарЃў, рЃЏрЃЉ рЃАрЃћрЃарЃЋрЃћрЃарЃў MYSQL, рЃљрЃАрЃћрЃЋрЃћ рЃњрЃљрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃБрЃџрЃў рЃюрЃљрЃЎрЃарЃћрЃЉрЃћрЃЉрЃў рЃЏрЃљрЃњ: Denver, OpenServer, XAMPP, WAMP, LAMP рЃЊрЃљ рЃљ.рЃе. рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЏрЃљрЃЌрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃф.
